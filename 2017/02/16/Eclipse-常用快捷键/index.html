Eclipse 常用快捷键

Posted on 2017-02-16

Ctrl + 1，快速修复
Ctrl + D，删除当前行
Ctrl + O，列举当前类中所有方法及属性
Ctrl + E，快速切换编辑器
Ctrl + Q，快速定位到最后一次编辑的地方
Ctrl + M，最大化或者最小化窗口
Ctrl + T，快速显示当前类的继承结构
Ctrl + L，定位于某行
Ctrl +F6，界面切换
Ctrl + /，为一行或者选择部分添加注释
Ctrl + Shift + R，快速查找资源
Ctrl + Shift + F，格式化某段代码或者整个Java文件
Alt + /，提供内容帮助
